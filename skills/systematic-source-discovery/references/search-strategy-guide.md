# Search strategy guide

Use concept families, synonyms, opposing terms, methods terms, cases, and author names. Document every search. Use backward, forward, and lateral citation chaining. Mark unverified leads clearly.

For systematic review mode, keep a PRISMA-compatible protocol snapshot: review question, databases, exact strings, dates, filters, screening counts, dedupe count, inclusion/exclusion rules, and exclusion reasons. Treat this as optional reporting-grade structure, not the default for ordinary book research.
