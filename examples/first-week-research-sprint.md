# First-week research sprint

Day 1: Use `scholarly-research-agenda` to define the project.

Day 2: Use `systematic-source-discovery` to build a search plan.

Day 3: Gather an initial source set and use `annotated-bibliography-builder`.

Day 4: Use `literature-review-mapper` to cluster the field.

Day 5: Use `argument-architecture` to build a thesis tree.

Day 6: Use `counterargument-peer-review` to stress-test it.

Day 7: Revise agenda, source plan, and chapter architecture.
