# Usage examples

## Build the full research workflow

```text
Use research-book-orchestrator. My book argues that [thesis]. Build a full research, writing, and verification workflow.
```

## Turn a broad idea into research questions

```text
Use scholarly-research-agenda. Turn this idea into a central research question, subquestions, scope boundaries, and evidence plan: [idea].
```

## Find sources systematically

```text
Use systematic-source-discovery. Build search families, Boolean query strings, database targets, inclusion/exclusion rules, and a search log for: [topic].
```

## Map the literature

```text
Use literature-review-mapper. Here are my sources and notes. Map schools of thought, debates, consensus, gaps, and how they affect my thesis.
```

## Audit claims

```text
Use claim-evidence-ledger. Extract the major claims in this draft and classify evidence status, citation need, risk, and safer wording.
```

## Challenge the thesis

```text
Use counterargument-peer-review. Challenge this thesis like a skeptical peer reviewer, then revise it into a stronger form.
```

## Audit citations

```text
Use citation-integrity-auditor. Check this chapter for unsupported claims, citation-source mismatch, quote/page needs, and fabricated-reference risk.
```
